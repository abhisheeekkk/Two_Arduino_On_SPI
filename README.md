  Pinout for Connection between Arduino Boards
  GND       -   GND
  D13       -   D13 (SCLK)
  D12       -   D12 (MISO)
  D11       -   D11 (MOSI)
  D10       -   D10 (CS)

  Please flash the SPI_Master arduini file onto one Arduino board.
  Another one onto the another Arduino board.
